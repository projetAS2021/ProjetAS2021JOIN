# Application Join.

Ce projet a pour vocation première de mettre les étudiants en situation de gestion de projet avec un effectif plus important et mettre en avant l’ensemble des savoirs étudiés liés à la conception et à la réalisation.
Nous proposons de concevoir et de réaliser une application de simulation de dépenses partagées. Celle-ci permettra aux utilisateurs de visualiser leurs dépenses faites ensemble de manière simple et précise, ainsi que de calculer la somme que les utilisateurs doivent donner l'un a l'autre selon un système de partage totalement modulable par les utilisateurs.

### Installation

#### Sur téléphone :

- Téléchargez et installez le fichier [``Join.apk``](https://github.com/projetAS2021/ProjetAS2021JOIN/raw/main/Join.apk) 
- Acceptez les conditions et installez l'application.

![image](https://user-images.githubusercontent.com/85195647/120457284-9e1acf80-c396-11eb-9f2b-bd03fa11783d.png)

Et voilà ! Vous avez installé Join.


#### Sur ordinateur :

- Executez la commande ``flutter doctor`` et vérifiez qu'il n'y a pas de problèmes liés a votre installation de Flutter.
- Clonez la branche main du projet avec git, si vous passez par la ligne de commande :   
``git clone https://github.com/projetAS2021/ProjetAS2021JOIN.git``
- Placez vous dans le fichier principal du projet : ``cd ProjetAS2021JOIN``
- Executez la commande ``flutter run``

## Fait avec :
* [Flutter](https://flutter.dev/) - Boîte a outils pour faire une interface, développé par Google
* [Dart](https://dart.dev/) - Langage utilisé pour coder l'interface
* [RESTClient](https://addons.mozilla.org/fr/firefox/addon/restclient/) - API REST pour la liaison avec la base de donnée
* [PHPMyAdmin](https://www.phpmyadmin.net/) - Système de gestion de base de données
* [Visual Studio Code](https://code.visualstudio.com/) - Editeur de textes
* [Android Studio](https://developer.android.com/studio?hl=fr) - SDK, émulateur Android et IDE
* [FileZilla](https://filezilla-project.org/) - Client FTP
* [PHP](https://www.php.net/) - Utilisé pour les requètes avec l'API
* [LAMP](https://doc.ubuntu-fr.org/lamp) - Serveur Web qui héberge la base de données
